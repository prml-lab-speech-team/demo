SENTENCE PROSODY GENERATION USING TEXT-CONDITIONAL SENTENCE PROSODY AUTOENCODER FOR NATURAL SPEECH SYNTHESIS

Authors : Sang-Hoon Lee, Hyun-Wook Yoon, Hyeong-Rae Noh, and Seong-Whan Lee

Abstract : Numerous attempts have been made to enable the text-tospeech (TTS) model to synthesize speech with a variety of speaking styles extracted from the input audio. Given that the input audio is an utterance of the particular sentence, the speaking style from the audio has specific sentence information. However, many use this speaking style as a prosody feature without removing text information. In this work, we introduce text-conditional sentence prosody autoencoder which can generate the appropriate sentence prosody for the text sequence to be synthesized by removing the text information of the input audio and predicting the text information of the target audio. We demonstrate that a synthesized speech with the generated sentence prosody features has not only speaker identity of the input audio, but also prosody for a particular sentence from different sentences of audio. Furthermore, we also evaluate zero-shot sentence prosody generation from the audio of an unknown speaker.
